🔗 CodeAlpha_Hangmangame
🔗 Hangman Word Prediction Game (Python)
A fun and interactive word guessing game built using Python, designed to improve logic, memory, and vocabulary skills through gameplay.

🔗 Features
Multiple Difficulty Levels
Easy Mode
Hard Mode

🔗 Word Length Variations
3-letter words
4-letter words
5-letter words

🔗 Random Word Selection
Words are randomly chosen each game for better replayability

🔗 Lives System
Limited attempts to make the game challenging and engaging

🔗 Interactive Gameplay
Step-by-step letter guessing system
Immediate feedback on correct/incorrect guesses

🔗 Menu Driven Interface
Easy navigation between game modes
Option to restart or exit anytime

🔗 Tech Stack
Python 
Random Module 
Functions & Loops 
Conditional Statements 

🔗 How It Works
Select game mode (Easy / Hard)
Choose word length
Guess letters step by step
Try to complete the word before lives run out
Win or try again! 
🎯 Objective
To build a simple yet engaging command-line game that demonstrates core Python programming concepts in a fun and practical way.
